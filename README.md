# GoldX

GoldX is an open-source algorithmic trading project focused on developing advanced TradingView indicators, MT5 Expert Advisors, and quantitative trading tools.

## Features
- TradingView Indicators
- MT5 Expert Advisors
- Backtesting Tools
- Quantitative Research
